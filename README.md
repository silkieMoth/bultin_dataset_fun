# R Built-in Dataset Visualization

### Purpose
R comes with 104 datasets upon installation, and while they are occasionally used for education they are usually overlooked even for this purpose (usually in favor of PalmerPenguins).
The purpose of this project is therefore to experiement with what can be done with them. It is therefore that this project has no specific aim, hypotheses, or target analyses. The only goal is for it to serve as a sandbox for the previously stated purpose.

### Thus Far.....

* Basic visualization of dataset: co2 (yes this is a different dataset)
